
# API

### API REST desarrollada con Node.js y Express

## Descripción
API REST simple con Node.js, Express y MongoDB que incluye operaciones CRUD para gestionar items.


## Instalación

1. Instalar dependencias:
```bash
npm install
```

2. Configurar variables de entorno:
Editar el archivo `.env` con tu configuración de MongoDB:
```
PORT=3000
MONGODB_URI=mongodb://localhost:27017/api_database
NODE_ENV=development
```

## Uso

### Modo desarrollo (con nodemon):
```bash
npm run dev
```

### Modo producción:
```bash
npm start
```
